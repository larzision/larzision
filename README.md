私聊指挥机器人帮你往群里发消息、艾特人、私聊传话，不用自己动手

## 能干啥

- **往群里发消息**：私聊说一声，我帮你润色后发到群
- **精准艾特人**：说昵称就行，我帮你找到人再@
- **私聊传话**：不想进群？我替你私聊别人
- **模糊搜群**：记不住群号？说群名关键词就行

## 怎么用

直接说人话，比如：

| 你说 | 我干嘛 |
|------|--------|
| 去游戏群问今晚打不打副本 | 搜"游戏"群，发「今晚打副本不～」 |
| 去摸鱼群告诉小李作业写完了没 | 搜"摸鱼"群，找到小李，@他问 |
| 私聊告诉小明明天开会 | 搜小明，私聊发「明天开会别忘了」 |

不用记群号不用记QQ号，说名字就行

## 工作原理

两个钩子 + 五个工具：

### 钩子
- **on_llm_request** — 检测到群/发/私聊/告诉等关键词就往system prompt塞操作规则
- **on_llm_response** — 工具返回结果精简美化，贴合人设风格

### 规则（注入到LLM提示词）
1. 直接调工具，不说自然语言
2. 失败输出错误信息
3. 没法调工具就发固定提示
4. 不提发了什么内容
5. 工具结果要改写成贴合人设风格的回复
6. 用自己的话重说润色，加语气词

### 工具
| 工具名 | 功能 |
|--------|------|
| `send_to_group_tool` | 往指定群发消息，可选@人 |
| `send_to_private_user` | 私聊发消息 |
| `get_group_id_by_name` | 群名模糊搜群号 |
| `get_user_id_by_name` | 群昵称搜QQ号 |
| `get_specified_group_members` | 拉群成员列表 |

## 注意

- 机器人必须在目标群里
- 仅私聊触发，群聊不理
- 纯文本消息，暂不支持发图
- 仅QQ平台
